## 1. 비전 (Vision)

> ### **"무역업계 44만이 매일 쓰는, 대한민국 대표 AX무역 레퍼런스가 된다."**

**비전 해석**:
본 사업의 수주·수행을 통해 단순한 SI 납품을 넘어, 향후 모든 공공 AX 사업의 제안서에 자랑스럽게 인용되는 "대한민국 대표 AI 무역 플랫폼 사례"가 되는 것이 우리의 지향점이다. 44만 멤버십·월 270만 방문자가 실제로 매일 사용하는 시스템이라는 사실 자체가 가장 강력한 레퍼런스 자산이며, 이는 회사의 향후 10년 공공 AX 사업 포트폴리오의 출발점이 된다.

**SWOT 근거**: SO-3(후속 확장 토대) + WO-3(본 사업=후속 레퍼런스) + O-D1(대표 AX 레퍼런스) 통합 메시지

---

## 2. 미션 (Mission)

> ### **"한국무역협회와 무역업계 44만 멤버십을 위해, 무역 도메인 격차를 AI/AX 기술의 압도적 우위로 역전한, 한 번도 본 적 없는 AX 플랫폼을 만든다."**

**미션 구성요소**:

| 요소 | 정의 |
|------|------|
| **핵심 고객 (Who)** | **1차**: 한국무역협회(KITA) — 발주처·운영주체 / **2차**: 무역업계 44만 멤버 + 월 270만 방문자 — 최종 사용자 / **3차**: 평가위원 — 수주 결정자 (Bid Phase 한정) |
| **제공 가치 (What)** | (1) Multi-Agent AI 20종·Data Lakehouse 기반의 AX 무역 디지털 혁신, (2) 무역 도메인 SME 5인+파트너 3사가 보증하는 도메인 정합성, (3) 44만 사용자가 실제 쓰는 실증된 시스템, (4) 회원사 ESG/CBAM 규제 대응 실질 지원 |
| **차별화 방식 (How)** | (1) **AI-First 백지 설계**: ISP 편견 없이 "현행 답습"이 아닌 "처음부터 AX"로 설계, (2) **실증 기반**: 8개 PoC/데모로 주장이 아닌 증명으로 설득, (3) **도메인 파트너십 자산화**: 외부 SME·파트너를 단순 자문이 아닌 사업 척추로 통합 |

**SWOT 근거**: SO-1(AI/AX 기술 우위) + SO-2(혁신 vs 답습 프레임) + WO-1(도메인 자산화) + WO-2(사용자 접근) 통합

## 3. 핵심가치 (Core Values)

본 사업의 모든 의사결정과 산출물에 일관되게 적용되는 4대 핵심가치이다.

| # | 핵심가치 | 정의 | 행동 지표 (실천 기준) |
|---|---------|------|----------------------|
| **CV-1** | **혁신 (Innovation)**<br>"처음부터 다시 그린다" | 기존 시스템·기존 사례·기존 관행에 갇히지 않고, AX 시대에 맞는 새로운 답을 처음부터 설계한다 | (1) 모든 To-Be 설계는 As-Is 분석 후 "백지 검토" 게이트 통과 / (2) 차별화 매트릭스 12개 포인트 정량 비교 의무 / (3) Multi-Agent·Lakehouse는 단순 도입이 아닌 KITA 특화 패턴으로 재설계 |
| **CV-2** | **실증 (Evidence-Based)**<br>"주장이 아닌 증명으로 말한다" | 모든 제안과 결과는 PoC·데모·데이터로 증명한다. "할 수 있다"가 아니라 "이미 보여줬다"로 말한다 | (1) 8대 영역 모두 PoC 또는 데모 영상 1건 이상 제출 / (2) Agent 응답 정확도 월간 1,000건 샘플링 검증 / (3) 모든 KPI는 검증 가능한 데이터 소스 명시 |
| **CV-3** | **신뢰 (Trust)**<br>"파트너십을 자산으로 키운다" | 무역 도메인 SME·컨소시엄 파트너·KITA 회원사 자문단을 단발성 자원이 아닌 장기 자산으로 대우한다. 약속한 일정과 품질은 단 한 번도 어기지 않는다 | (1) SME 5인의 분기 참여 시간 인당 40h 이상 보장 / (2) 마일스톤 적기 달성률 100% (지연 0건) / (3) 파트너 SLA 준수율 95% 이상 / (4) 회원사 자문단 분기 1회 사용성 인터뷰 |
| **CV-4** | **집중 (Focus)**<br>"이길 영역에 자원을 몰빵한다" | 우리의 강점이 직접 작동하는 영역(Trade Digital Brain, Trade Agent, 대시보드, ESG)에 자원을 압도적으로 투자하고, 도메인 의존도 높은 영역은 파트너 주도 구조로 명확히 분리한다 | (1) 자원 배분 70% 이상을 AI/AX 핵심 4개 영역에 집중 / (2) 도메인 영역은 컨소시엄·SME 주도 구조 명시 / (3) 모든 결정에 "이것이 SO-1을 강화하는가?" 질문 의무 |

**SWOT 근거**:
- CV-1 (혁신) ← SO-2 "현행 답습 vs AX 혁신 프레임"
- CV-2 (실증) ← ST-1 "PoC·데모로 정보격차 무력화"
- CV-3 (신뢰) ← WO-1 "SME·컨소시엄 도메인 자산화" + ST-4 "컴플라이언스 사전 차단"
- CV-4 (집중) ← WT-1 "선택과 집중"

## 4. 비전·미션·핵심가치 정합성 검증

### 4.1 OKR 연계 (무엇을 달성할 것인가)

| 비전·미션 요소 | 연계 Objective | 핵심 KR |

| 비전 "44만이 매일 쓰는" | **O-D2** Multi-Agent 안착 + **O-D3** Lakehouse 의사결정 엔진 | KR-D2.4(위젯 30종/주간 5개+), KR-D3.4(ESG 회원사 300+) |
| 미션 "AI/AX 기술 압도적 우위" | **O-B1** 압도적 기술 차별화 | KR-B1.1(평가 76+), KR-B1.3(차별화 12개+), KR-B1.4(데모 8건) |
| 미션 "도메인 격차 역전" | **O-B2** 약점→자산 전환 | KR-B2.1(SME 5인+), KR-B2.2(컨소시엄 3사), KR-B2.3(인터뷰 15건+) |
| 미션 "처음부터 만든다" (AI-First) | **O-B3** ISP 정보 비대칭 좁히기 | KR-B3.4(To-Be 아키텍처 청사진) |
| CV-1 혁신 + CV-4 집중 | **O-D4** 팀·회사 동반성장 | KR-D4.2(재사용 자산 50건+), KR-D4.3(후속 수주 2건+) |

### 4.2 SWOT 근거 (왜 이 비전이 적합한가)

| 비전·미션·가치 | 활용한 SWOT 요소 | 적합성 논거 |
|----------------|------------------|------------|
| 비전 (대표 AX 레퍼런스) | O5 후속 확장 + O7 44만 사용자 + S7 자산화 | 기회·강점의 정확한 교차점 — Top 우선순위 SO-3 직접 구현 |
| 미션 (AI/AX 역전) | S1 AI/Agent + S2 Lakehouse + O1 AX 수요 + O2 기술 80% | Top 1순위 전략 SO-1을 미션 문장으로 압축 |
| CV-1 혁신 | S4 객관 시각 + O3 ISP 보수성 | 우리만의 구조적 우위(편견 없음)를 가치 차원에서 보존 |
| CV-2 실증 | S6 PoC 캐파 + T1 정보 비대칭 | 정보격차라는 위협을 "실증력"이라는 가치로 정면 대응 |
| CV-3 신뢰 | W2 도메인 부재 + W3 KITA 관계 부재 | 약점을 외부 파트너십으로 보완하되, 일회성이 아닌 자산화 의지를 가치로 선언 |
| CV-4 집중 | W4 도메인 인력 0 + T2 경쟁사 도메인 우위 | "전선을 좁혀 이긴다" — WT-1 회피 전략의 가치 차원 표현 |

### 4.3 BSC 4대 관점 구현 경로 (어떻게 측정·실현하는가)

| 관점 | 비전 구현 경로 | 핵심 KPI (BSC) |
|------|---------------|----------------|
| **학습·성장** (출발점) | CV-3 신뢰 + CV-4 집중 → SME 자산화·도메인 인증 인력 양성 | LG1(SME 누적), LG2(인증 인력 30+), LG3(인사이트 라이브러리), LG5(파트너십 생태계) |
| **내부 프로세스** (변환기) | CV-1 혁신 + CV-2 실증 → 제안서 품질·Agent/Lakehouse SLA·SME 검수 게이트 | IP1(요구사항 A+ 75%), IP2(PoC 8/8), IP5(Agent SLA), IP6(결함 처리) |
| **고객** (검증대) | 미션 "44만을 위해" → 평가위원→KITA→회원사 3계층 만족 | C1(평가 76+), C3(KITA 만족도 4.5+), C4(사용자 60%), C5(NPS +20) |
| **재무** (최종 성과) | 비전 "대표 레퍼런스" → 수주 + 수익률 + 후속 파이프라인 | F1(수주 계약액), F2(수익률 12%+), F4(후속 100억+) |

전략맵의 "학습·성장 → 내부 프로세스 → 고객 → 재무" 화살표가 곧 미션 "도메인 격차를 AI 기술 우위로 역전"의 시각적 증거이다.

### 4.4 핵심 메시지 정렬 (제안서 ↔ 비전·미션)

본 비전·미션은 제안서 핵심 메시지로 그대로 사용 가능하도록 설계되었다:

```
[제안서 표지·헤드라인]
"현행을 다시 만들지 않습니다. AX를 처음부터 만듭니다."
       ↑
       └── 미션의 "처음부터 만든다" + CV-1 혁신의 직접 표현

[제안서 3가지 증명]
1. AI/Agent/Lakehouse 8건 PoC 실증     ← CV-2 실증 + SO-1
2. 무역 도메인 SME 5인 + 파트너 3사     ← CV-3 신뢰 + WO-1
3. 44만 사용자가 실제 쓰는 시스템       ← 비전 + WO-2/SO-3
```

## 5. 비전·미션의 시간축 운용

### 5.1 Bid Phase (2026-05 ~ 2026-06)

- **운용 중심**: 미션 — 수주를 위한 핵심 메시지 송출
- **활용 지점**: 제안서 표지·요약·결론 / 제안 설명회 / 평가위원 Q&A
- **체크 질문**: "이 페이지는 미션을 강화하는가?" / "이 표현이 CV-1·CV-2를 위반하지 않는가?"

### 5.2 Delivery Phase (2026-07 ~ 2027-06)

- **운용 중심**: 비전 — 사업 수행의 북극성
- **활용 지점**: 분기 만족도 리뷰 / 마일스톤 회고 / 백서·레퍼런스 자료화
- **체크 질문**: "이 결과물이 '대표 레퍼런스'에 등록될 수준인가?" / "44만이 매일 쓰게 되는가?"

### 5.3 핵심가치의 일관 운용

- **CV-1~CV-4는 Bid·Delivery 양 Phase 일관 적용**
- 분기 OKR 체크인 시 "어떤 결정이 어떤 핵심가치를 강화/약화시켰는가" 항목 의무 회고
- 신규 의사결정 발생 시 CV-4(집중) 질문을 1차 게이트로 사용: "이것이 SO-1을 강화하는가?"
