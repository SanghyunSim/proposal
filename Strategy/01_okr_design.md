# 01. OKR 설계서 — KITA 차세대 AX무역플랫폼 구축사업

> 작성일: 2026-05-19
> 작성자: okr-designer 에이전트
> 검증 기준: QSIM(Objective) + SMART-V(Key Result)
> 전략 핵심: "도메인 경험의 격차를 AI 기술력의 압도적 우위로 역전한다"

---

## 1. 조직 개요 및 OKR 설계 전제

### 1.1 조직 상황 요약

- **수행 주체**: 우리 회사 (AI/클라우드/데이터 플랫폼 강점)
- **대상 사업**: 한국무역협회(KITA) 차세대 AX무역플랫폼 구축 (총 187개 요구사항, 8대 영역)
- **평가 비중**: 기술 80 / 가격 20 → 기술 차별화가 사실상 결정변수
- **핵심 제약**: 무역 도메인 경험 부재 + 경쟁사 ISP 직접 수행

### 1.2 OKR 설계 원칙

1. **2단계 OKR 체계**: 수주(Bid) Phase + 수행(Delivery) Phase 분리, 각 Phase에 독립 Objective
2. **Stretch Goal**: 0.7 달성을 성공 기준으로 설정 (도전적 목표)
3. **약점 보완 OKR 명시**: 도메인 갭, ISP 정보 비대칭을 KR 레벨에서 직접 다룸
4. **수치형 KR 70% / 마일스톤 30% / 바이너리 0%** 원칙 준수
5. **회사 KR ↔ 부서 KR 수직 정렬**, 부서 간 상충 KR 사전 차단

### 1.3 OKR 주기 및 체크인 리듬

| Phase | 기간 | 주기 | 체크인 |
|-------|------|------|--------|
| Bid Phase | 2026-05 ~ 2026-06 (8주) | 2주 단위 미니 OKR | 주간 체크인 (월요일 09시) |
| Delivery Phase | 2026-07 ~ 2027-06 (12개월) | 분기 단위 OKR | 격주 체크인 + 분기 리뷰 |

## 2. Bid Phase OKR (2026-05 ~ 2026-06): 수주 전략

### 2.1 Company-Level Objective 1 (Bid)

**O-B1. "압도적 기술 차별화로 KITA의 첫 번째 선택지가 된다"**

> 가격이 아닌 기술로 승부하는 발주 구조(80:20)를 활용해, ISP 수행사의 "현행 답습"과 명확히 대비되는 AX 혁신 제안서를 만들어 기술평가 1위를 달성한다.

| ID | Key Result | 시작값 | 목표값 | 검증 방법 |
|----|-----------|--------|--------|----------|
| KR-B1.1 | 기술평가 자체 모의평가 점수를 70점에서 **76점 이상**으로 달성 (외부 전문가 3인 블라인드 채점, 6월 1주차) | 70 | 76+ | 외부 평가위원 평가표 |
| KR-B1.2 | RFP 187개 요구사항 중 **'우수(A+)' 평가 응답 비율을 50%에서 75%로 향상** (5월말 1차→6월초 최종) | 50% | 75% | 내부 QA 체크리스트 |
| KR-B1.3 | Multi-Agent / Data Lakehouse 영역에서 경쟁사 대비 **차별화 포인트 12개 이상** 제안서에 명시 (각 포인트당 정량 비교 근거 포함) | 0 | 12+ | 차별화 매트릭스 문서 |
| KR-B1.4 | 8대 영역 모두에 대해 **실증 가능한 PoC 또는 데모 영상 1건 이상** 제안서 첨부 (총 8건) | 0 | 8 | 제안서 부록 데모 인덱스 |
| KR-B1.5 | **가격평가 16점 이상 확보** (20점 만점) — 최종 입찰가를 예산 한도 대비 **0.92~0.97 범위**로 산정하여 가격경쟁력 확보 (SWOT WT-2 대응, BSC F3 연계) | 0 | 16+ / 0.92~0.97 | BoQ 산정 모델 + 가격 평가 시뮬레이션 |

**QSIM 자가 검증**:
- Q(Qualitative): ✓ 숫자 없음, "첫 번째 선택지" 방향 제시
- S(Strategic): ✓ "기술 차별화로 도메인 격차 역전" 전략 방향과 직접 연결 + 가격 20점 부문도 흡수
- I(Inspiring): ✓ "압도적", "첫 번째" 동기부여 언어
- M(Memorable): ✓ 한 문장, 명확

**QSIM 자가 검증**:

- S: ✓ "도메인 갭 보완"이라는 전략 약점 정면 대응 + WT-3·WT-4 위협까지 흡수
- I: ✓ "전환한다"는 역전 메시지
- M: ✓ 핵심 단어 "약점 → 자산"

## 3. Delivery Phase OKR (2026-07 ~ 2027-06): 수행 전략

### 3.1 Company-Level Objective 1 (Delivery)

**O-D1. "KITA가 자랑하는 대한민국 대표 AX무역 레퍼런스를 만든다"**

> 단순 납품을 넘어, 향후 회사의 공공 AX 사업 모든 제안서에 대표 사례로 인용될 수 있는 수준의 결과물을 만든다.

| KR-D1.2 | 8대 영역 모두 **계약 마일스톤 100% 적기 달성** (지연 0건) | 0 | 8/8 | 마일스톤 체크리스트 |
