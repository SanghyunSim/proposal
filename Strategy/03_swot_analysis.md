# 03. SWOT 분석서 — KITA 차세대 AX무역플랫폼 구축사업 (제안사 관점)

> 작성일: 2026-05-19
> 작성자: swot-specialist 에이전트
> 분석 방법론: 내부분석(7S + VRIO) / 외부분석(PESTLE) / TOWS 매트릭스
> 우선순위 평가 기준: 영향도 30% / 실행가능성 25% / 시급성 20% / 리스크 15% / 자원효율 10%
> 전략 핵심: "도메인 경험의 격차를 AI 기술력의 압도적 우위로 역전한다"

---

## 1. 분석 범위

- **분석 대상**: 우리 회사(제안사) — KITA 차세대 AX무역플랫폼 구축사업 수주·수행 관점
- **분석 시점**: 2026.05 (공고 2026-05-06, 제출 마감 2026-06-12, Bid Phase 진행 중)
- **분석 범위**:
  - 단기(8주): 제안서 작성 및 수주
  - 중기(12개월): 수주 후 사업 수행 (2026-07 ~ 2027-06)
- **이중 고객 구조**: KITA(직접 발주처) + 무역업계 44만 회원사(최종 사용자)
- **데이터 한계**: 경쟁사 구체 정보는 "제한적 데이터 기반" — 일반 SI 시장 통념 + RFP·OKR 입력 기반 추정

## 2. SWOT 매트릭스

### 2.1 Strengths (강점) — 7S/VRIO 기반

|---|------|-----------|------------|
| **W1** | **[핵심] 유사 대형 SI 프로젝트 수행 경험 부족** | 동급 규모(187개 요구사항, 8대 영역, 12개월) 통합 SI 레퍼런스가 경쟁사 대비 열위 — 특히 공공 부문 SI 메이저 대비 | 단기: 컨소시엄·하도급 LOI로 보완 / 중기: 본 사업 자체가 레퍼런스 |
| **W2** | **[핵심] 무역 도메인 경험 전무** | 무역 프로세스(수출입·통관·신용장·원산지), 무역통계, 관세·CBAM/CSRD 규제 등 도메인 인력·자산 부재 | 단기: SME 5인 + 도메인 파트너 3사로 보완 (KR-B2.1, KR-B2.2) / 중기: 사내 인증 30명 (KR-D4.1) |

| **W4** | **무역 도메인 상주 인력 0명** | 관세사, 무역영어 자격, 무역업 실무 경력자 사내 풀 없음 — TF 구성 시 외부 영입 필수 | 6월까지 채용 비현실적 → 컨소시엄·자문에 단기 의존 |
| **W5** | **공공 평가위원 친화적 산출물 표준 경험 부족** | 공공 SI 제안서 특유의 트레이서빌리티 매트릭스·심사위원 가독성 최적화 노하우 상대적 열위 | 외부 제안 컨설팅 + 모의평가 2회(KR-Q1 연계)로 보완 |
| **W6** | **가격 전략·원가 구조 검증 미흡** | 무역 도메인 작업량 산정 정확도 낮음 → 가격 점수 20% 영역에서 적정선 산정 리스크 | 컨소시엄 단가 협의 + 유사 사업 벤치마킹으로 보완 |
| **W7** | **현장 사용자 접근성 부족** | KITA 현업 인터뷰·관찰 기회 제한 — 경쟁사는 ISP 단계에서 이미 다수 확보 | 공개 채널(컨퍼런스·회원사 행사) 활용 + 자문단 우회 접근 |

### 2.4 Threats (위협) — PESTLE 기반

| # | 위협 | 근거/데이터 | 대응 시급성 |
|---|------|-----------|------------|
| **T1** | **[핵심] 경쟁사가 ISP 직접 수행 → 정보 비대칭** | 경쟁사는 현행 시스템 구조, KITA 내부 요구, 현업 담당자 인맥, 기술 상세를 모두 사전 보유 → "현행 이해도" 평가 영역 절대 우위 | **최우선(즉시)** — Bid Phase 8주 안에 격차 좁혀야 |
| **T2** | **경쟁사의 무역 도메인 전문성 + KITA 기존 관계** | 통상 KITA 사업 단골 SI사는 무역·통상 도메인 수행 누적 사례 다수, 발주처 신뢰도 우위 | **즉시** — 도메인 보완 자산(SME·컨소시엄) 시급 확보 |
| **T3** | **평가 위원단의 레퍼런스 중심 판단 경향** | 공공 평가위원은 통상 "유사 사업 수행 실적"을 안전한 판단 근거로 선호 → 신규 진입자 불리 | **즉시** — 레퍼런스 재해석(KR-B2.4) + 차별화 매트릭스(KR-B1.3) |
| **T4** | **187개 요구사항의 광범위성 → 도메인 지식 없는 팀의 누락 위험** | 무역 도메인 용어·관행에 대한 무지로 요구사항 해석 오류, 응답 누락, 트레이서빌리티 결함 가능성 | **즉시** — SME 검수 단계 필수, QA 사이클 2회 이상 |
| **T5** | **경쟁사의 가격 덤핑 가능성** | 도메인 우위 경쟁사가 후속 사업 확장을 노리고 본 사업을 저가 수주할 경우 가격 20점에서 큰 격차 | 단기 — 가격 전략 보강 필요 (W6과 연계) |
| **T6** | **Multi-Agent / LLM 기술의 빠른 진화로 인한 아키텍처 진부화 리스크** | 12개월 수행 기간 중 LLM/Agent 표준 급변 가능, 초기 설계가 종료 시점에 진부화될 위험 | 중기 — 모듈러·교체 가능 아키텍처 설계로 헤지 |
| **T7** | **기밀정보·개인정보 처리 컴플라이언스 위반 리스크** | 무역업계 기밀 정보(거래정보·계약·원가) 다수 처리 → 단 1건 사고로 레퍼런스 전면 무력화 | 중기 — KR-Q1.2, KR-Q1.4로 사전 차단 |
| **T8** | **공공 발주 일정 지연 또는 요건 변경 가능성** | 공고 후 변경공고·재공고·요건 추가 가능성 (공공 사업 통상 리스크) | 단기 — 컨틴전시 시나리오 사전 준비 |
| **T9** | **글로벌 빅테크의 AI 무역 플랫폼 진입 가능성** | Microsoft/Google/AWS 산업 특화 AI 솔루션이 공공 영역 진입 시 기술 우위 약화 가능 | 중기 — 도메인 특화 + 공공 운영 노하우로 차별화 |

- 수주 시 매출 확보(SO-1), 후속 사업 파이프라인(SO-3, KR-D4.3) → 핵심 KPI
- **WT-2 가격 전략은 BSC 재무 관점에서 별도 KPI화 권고**: "가격 점수 / 출혈입찰 여부 / 원가율"
- T5(가격 덤핑) 위협에 대응한 손익분기점·출혈선 사전 설정 필수

**고객 관점**:
- 이중 고객(KITA + 44만 회원사) 구조 → BSC에 2계층 KPI 분리 권고
  - KITA 직접 만족도: KR-D1.1 (4.5/5)
  - 회원사 사용자 채택: KR-D1.4 (60%)
- WO-2(회원사 자문단 20명) → 고객 관점 선행지표로 KPI화

**내부 프로세스 관점**:
- SO-1·ST-1·ST-3·ST-4 모두 내부 프로세스 강화 전략 → 직접 KPI화
- **WT-3 SME 검수 게이트** → "도메인 검수 통과율"을 프로세스 KPI 신설
- Multi-Agent 운영 SLA(KR-D2.3), Lakehouse 가용성(KR-D3.2) 핵심

**학습·성장 관점**:
- WO-1·WO-3·KR-D4.x 직접 매핑
- W4(무역 도메인 인력 0명) → 학습·성장의 최우선 격차
- SO-3(후속 확장) → 자산화 라이브러리·재사용률 KPI

### 8.2 BSC 사각지대 후보
